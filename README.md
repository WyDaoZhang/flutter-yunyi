# 云弈ai

该仓库基于 Flutter，适用于开发跨平台（Android、iOS、Web）应用。Flutter 作为 Google 推出的 UI 框架，能够统一编写一套代码即可发布到多平台，因此 flutter-yunyi 项目适合需要高度一致性体验的多端应用场景，比如：企业运营平台、资讯发布平台、工具类 App、Web 小应用等。

## 技术组成
主要语言：Dart（约 69.2%），用于业务逻辑、UI 构建及应用架构。
辅助语言：HTML（22.7%），暗示部分页面或功能具有 Web 端拓展或嵌入式 Web 模块。
原生扩展：C++、Swift、C，涉及原生插件开发或平台底层能力（例如设备接口、性能优化）。
工程配置：CMake，用于跨平台原生库的组织与构建。
